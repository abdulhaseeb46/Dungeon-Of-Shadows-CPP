# Dungeon of Shadows

A console-based RPG game developed in C++ using Object-Oriented Programming concepts.

## Features

- Multiple playable classes
  - Warrior
  - Mage
  - Archer
  - Rogue

- Enemy system
- Boss battle
- Save & Load game
- Weapon inventory
- Character leveling
- XP system
- Health bars
- ASCII Art
- Colored console interface

## OOP Concepts Used

- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

## Technologies

- C++
- Windows Console
- Object-Oriented Programming

## How to Run

Compile using g++ or Visual Studio.
